# Zo Garden Documentation

This `content/` folder is the canonical documentation surface for the Zo Garden monorepo. It is rendered by [Quartz 4](https://quartz.jzhao.xyz/) as a static site.

## Structure

- [Roadmap](./roadmap/ROADMAP.md) — canonical milestone sequencing and status.
- [Milestones](./milestones/) — active, retroactive, and completed milestone artifacts.
- [Operations](./operations/) — runbooks, execution logs, scripts, and operational artifacts.
- [Architecture](./architecture/) — `ARCH-*` decision and system-shape documents.
- [Technical Debt](./technical-debt/) — `TD-*` risk and remediation records.
- [Definitions](./_definitions/) — artifact definitions and authoring conventions.
- [Getting Started](./GETTING_STARTED.md) — onboarding and contribution flow.
- [Project Status](./PROJECT_STATUS.md) — transient recent-changes snapshot.

## Quick start

- For roadmap and delivery status: open [ROADMAP](./roadmap/ROADMAP.md).
- For current execution work: open [Milestones](./milestones/).
- For runtime validation: open [Operations index](./operations/index.md).
- For onboarding and conventions: open [Getting Started](./GETTING_STARTED.md).

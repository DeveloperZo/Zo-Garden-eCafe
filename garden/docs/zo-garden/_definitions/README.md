# Definitions

Canonical artifact definitions for the Zo Garden doc surface. Each file below is the authoring contract for one doc type — required sections, file-name convention, and where it lives.

- [`architecture.md`](./architecture.md) — `ARCH-###`
- [`milestone.md`](./milestone.md) — `M-###` and `RM-###`
- [`operations.md`](./operations.md) — `OPER-###`

- [`technical-debt.md`](./technical-debt.md) — `TD-###`
- [`video.md`](./video.md) — `VID-###` (short-form and long-form video scripts)

Use these as the contract whenever you create or update a doc.
